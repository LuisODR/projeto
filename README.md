# 🌿 CompostFlow — Resíduos que Geram Valor

O **CompostFlow** é uma plataforma digital inovadora projetada para gerenciar o ciclo de vida de resíduos orgânicos, transformando passivos ambientais em ativos valiosos através da compostagem inteligente e da economia circular.

---

## 🚀 Funcionalidades Principais

* **E-commerce Sustentável:** Loja integrada para venda de adubos orgânicos, biofertilizantes e equipamentos de compostagem.
* **Gestão de Lotes:** Dashboard para monitoramento de lotes de compostagem em tempo real (Parâmetros de pH, umidade e temperatura).
* **Logística Inteligente:** Mapa interativo (Leaflet.js) para localização de pontos de coleta e parceiros.
* **Análise de Dados:** Visualização de métricas de impacto ambiental e financeiro através de gráficos dinâmicos (Chart.js).
* **Checkout Customizado:** Sistema de finalização de compra em múltiplas etapas focado na experiência do usuário.
* **Design Premium:** Interface responsiva com tipografia refinada e animações suaves, utilizando variáveis CSS para um sistema de cores consistente.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias modernas de front-end, focando em performance e interatividade:

| Tecnologia | Finalidade |
| :--- | :--- |
| **HTML5** | Estruturação semântica do conteúdo. |
| **CSS3** | Estilização avançada (Flexbox, Grid, Variáveis e Animações). |
| **JavaScript (ES6+)** | Lógica de negócio, manipulação de DOM e estados do carrinho. |
| [**Leaflet.js**](https://leafletjs.com/) | Mapas interativos e geolocalização de parceiros. |
| [**Chart.js**](https://www.chartjs.org/) | Renderização de gráficos de monitoramento e impacto. |
| [**Google Fonts**](https://fonts.google.com/) | Tipografia (Playfair Display e DM Sans). |

## 🎨 Design System

O projeto utiliza uma paleta inspirada na natureza e elementos terrosos:
* `--gd` (#1a3a1f): Verde Escuro (Deep Forest) - Cor primária.
* `--ga` (#4caf50): Verde Ativo (Natural Green) - Destaques e CTAs.
* `--el` (#c9a227): Amarelo Terra (Earth Gold) - Acentos e alertas.
* `--cream` (#f7f3ec): Fundo Suave (Nature Cream) - Conforto visual.

## 📂 Metodologia Aplicada

Este projeto utiliza conceitos de **Design Thinking** e a técnica **SCAMPER** para otimizar serviços de compostagem urbana, focando em:
* **Substituição** de processos manuais por monitoramento digital.
* **Combinação** de e-commerce com educação ambiental.
* **Adaptação** de modelos de logística reversa para resíduos orgânicos.
